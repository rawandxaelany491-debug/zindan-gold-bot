# بۆتی ستراتیژی SNRZ (تایبەت)

بۆتێکی Telegram کە بەپێی ستراتیژی **SNRZ** (دروستکراوی خۆت) وەڵامی پرسیار
دەداتەوە و شیکاری چارت (وێنە) دەکات. **تەنها خاوەن و ئەو کەسانەی خاوەن
ڕێگەیان پێدابێت دەتوانن بەکاری بهێنن** — بۆتەکە بۆ گشتی خەڵک نییە.

## ١. ئامادەکردنی پێداویستیەکان

### أ) دروستکردنی بۆتی Telegram
1. لە تلیگرام بگەڕێ بۆ **@BotFather**
2. فەرمانی `/newbot` بنێرە و ناوێک بۆ بۆتەکەت هەڵبژێرە
3. تۆکنێکت پێدەدرێت (شتێک وەک `123456789:ABCdefGhIJKlmNoPQRstuVWXyz`)
   — ئەمە دەخەیتە `TELEGRAM_BOT_TOKEN` لە فایلی `.env`

### ب) وەرگرتنی OpenAI API Key
1. بڕۆ بۆ https://platform.openai.com/api-keys
2. هەژمارێک دروست بکە (ئەگەر نەتهەیە) و کلیلێکی API دروست بکە
3. ئەمە دەخەیتە `OPENAI_API_KEY` لە فایلی `.env`
4. دڵنیابەرەوە هەژمارەکەت بڕی پارەی تێدا هەیە (Billing) بۆ ئەوەی کلیلەکە کاربکات

### ج) وەرگرتنی ئایدی تلیگرامی خۆت
1. لە تلیگرام بگەڕێ بۆ **@userinfobot** و `/start` بنێرە، یان
2. دوای ڕاخستنی بۆتەکە فەرمانی `/id` بنێرێت (بەڵام سەرەتا پێویستە ئایدیەکەت
   وەک OWNER دابنێیت پێش ڕاگەیاندن — بۆیە باشترە یەکەم ڕێگە بەکاربهێنیت)
3. ئەمە دەخەیتە `OWNER_TELEGRAM_ID` لە فایلی `.env`

## ٢. دامەزراندن

```bash
cd telegram-trading-bot
pip install -r requirements.txt --break-system-packages
cp .env.example .env
```

دواتر فایلی `.env` بکەرەوە و نرخە ڕاستەقینەکانی خۆت دابنێ.

## ٣. ڕاخستنی بۆتەکە

```bash
python bot.py
```

بۆتەکە دەستپێدەکات و لە تلیگرامەوە بەردەستە. دەتوانیت بۆتەکە لەسەر
سێرڤەرێک (وەک VPS) بەردەوام ڕابگرێت تاکو هەموو کات کار بکات.

## ٤. چۆنیەتی بەکارهێنان

### وەک خاوەن (تۆ):
- `/start` — دەستپێکردن
- هەر پرسیارێک بنووسە دەربارەی SNRZ — وەڵامی دەداتەوە
- وێنەی چارتێک بنێرە (لەگەڵ caption یان بەبێی) — شیکاری بۆ دەکات
- `/reset` — پاککردنەوەی مێژووی گفتوگۆکەت
- `/allow <user_id>` — ڕێگەدان بە کەسێکی تر بۆ بەکارهێنانی بۆتەکە
- `/deny <user_id>` — سەندنەوەی ڕێگە لە کەسێک
- `/users` — بینینی لیستی هەموو کەسانی ڕێگەپێدراو

### بۆ ڕێگەدان بە کەسێکی تر:
1. داوابکە ئایدی تلیگرامی خۆی بنێرێت بۆت (بە فەرمانی `/id` لای خۆی —
   بەڵام تا ڕێگەی پێنەدرابێت، بۆتەکە تەنها ئایدیەکەی نیشانی دەدات و
   وەڵامی پرسیارەکانی نادات)
2. تۆ فەرمانی `/allow <ئایدییەکە>` بنێرە
3. ئیتر ئەو کەسە دەتوانێت بۆتەکە بەکاربهێنێت

## تێبینی گرنگ

- ستراتیژیەکە لە فایلی `strategies.py` دایە — دەتوانیت هەر کاتێک
  بیگۆڕیت یان زیادی بکەیت.
- بەکارهێنانی OpenAI API نرخی هەیە بەپێی بڕی بەکارهێنان (تکایە سەیری
  نرخنامەی OpenAI بکە لە platform.openai.com).
- لیستی ڕێگەپێدراوان لە فایلی `allowed_users.json` پاشەکەوت دەکرێت،
  خۆکارانە دروست دەبێت.
